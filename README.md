# Code generator

It just generates random codes to be used as you wish. Here's an example of code: `00124b00188a7f6810`.

## How to run

![rust] ![cargo]

| Description | Command |
| :--- | :--- |
| Run tests | `cargo test`
| Run for development | `cargo run` |
| Generate the binary | `cargo build --release` |
